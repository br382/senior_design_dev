These .py files were all packaged into seprate .apk files in ./bin/*.apk
Their success/fail/runtime logs were saved at ./bin/*.log

For developement useage, 'threading' should be used.
The import 'thread' is the basic functionality API that 'threading' is built with.

Note however in most cases the constructor and methods of 'threading' and 'multiprocessing' are identical in operation and spelling.
